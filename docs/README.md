# My Component Library

欢迎使用我的组件库！这是一个基于 Element UI 的自定义组件库。

## 快速开始

1. 安装依赖：

   ```bash
   npm install
   ```

2. 运行文档：

   ```bash
   npm run docs:dev
   ```

3. 浏览器中访问 `http://localhost:8080` 查看文档。
